## CAI DAT VA SU DUNG

### Yeu Cau
*   Windows 10 hoac 11
*   Python 3.10+
*   RAM 4GB+

### Cai Dat
1.  **Clone project**:
    `git clone https://github.com/vandang890615/vietlott.git`
2.  **Cai dat thu vien**:
    `pip install -r requirements.txt`
3.  **Chay phan mem**:
    Chay file `MO_PHAN_MEM.bat`
